# InventarioTech

Sistema de inventario desarrollado en C# con Windows Forms .NET Framework.

## Funcionalidades
- Agregar productos
- Buscar productos por código
- Editar productos
- Eliminar productos
- Limpiar campos

## Tecnologías
- C#
- Windows Forms
- .NET Framework
